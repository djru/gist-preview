Frequently Asked Questions

What is Gist Preview?

Gist Preview is a service that allows you to view rendered versions of HTML gists (which, surprisingly, Github doesn't already do).

How can I access the raw version of my gist?

You can access the raw version of the gist by clicking the button with the brackets in the uppermost right corner of the gist.

What happens if I try to preview a gist that does not contain html?

The code from the gist will be treated as plain text.

Can I use this service to host my static website?

No one is stopping you from using Gist Preview to host your website, but Github Pages is a much better alternative.
